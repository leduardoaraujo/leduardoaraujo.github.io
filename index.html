GAV Insights

copiloto corporativo

Copiloto que conversa em linguagem natural sobre o ecossistema analítico da empresa: seleciona relatórios, consulta o Power BI e mostra resultados — respeitando regras de negócio e permissões antes de responder. Seu motor de retrieval é open-source, o OpenContext.

Python · React · Power BI · Fabric · PostgreSQL · SQL Server · MCP · RAG · DAX

Framework ETL para ingestão de APIs, com pipelines declarados em YAML: chamadas assíncronas, transformação de JSON e entrega simultânea para múltiplos destinos.

Python · asyncio · FastAPI · PostgreSQL · DuckDB · Airflow

Python · PaddleOCR · OpenCV · Pydantic · Typer · pytest
